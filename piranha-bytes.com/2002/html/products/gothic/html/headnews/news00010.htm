27.03.2001: Neue Gothic Demo


Diese Demo liegt zur Zeit exclusiv auf Gamesmania.de.
Die Demo ist etwa 230 MB groß und umfaßt einen Teil der Oberwelt.
Wer also noch nichts von Gothic gesehen hat oder einfach nur die erste Demo gespielt hat, wird überrascht werden. Denn diese Demo ist mit der Finalen Version von Gothic identisch.
So werden sowohl die Voodoo 3 als auch ATI KArten, für die ATI noch neuere Treiber veröffentlicht, unterstützt.
