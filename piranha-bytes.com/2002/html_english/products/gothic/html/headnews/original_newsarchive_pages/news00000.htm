27/03/2001: New Gothic Demo


The demo is currently exclusively available at Gamesmania.de.
It takes up about 230 MB and contains part of the Upperworld.
So, if you haven't seen anything of Gothic yet or have only played the first demo so far, you'll be surprised. The new demo is identical to the final version of Gothic.
It supports Voodoo 3 and ATI cards (which ATI will be providing updated drivers for).
