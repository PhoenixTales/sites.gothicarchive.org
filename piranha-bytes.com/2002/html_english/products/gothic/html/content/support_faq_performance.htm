Performance Tuning:

General

Close all programs you have running in the background. Tasks running parallel may harm the performance and stability 
of GOTHIC. Make sure that you have defragmented your hard drive. Fragmented hard drives slow down the access to the hard drive, which may hinder the loading process. If your hard drive has UDMA capability (as all later models should), you should activate it in your Control Panel.

Do not compress you hard drive or use systems such as DriveSpace, which compress data on your hard drive. Otherwise the data will have to be decompressed again while loading.

Here is a simple rule regarding the influence certain hardware components may have on the performance:

Memory. Having 256 MB rather than 128 MB should noticeably improve the performance
CPU: Use at least a K6-400 and an adequate amount of memory to experience good gameplay with a good video card
Video card: More texture memory is more important than a high fillrate

Experiment with the options in the Video menu to find the ideal compromise between performance and video quality.

A good website which supplies additional tuning tips: www.tweakfiles.com.
An excellent reference tool to determine your PC's 3D performance is 3D Benchmark 3DMark 2000,
which is available at www.madonion.com.
If this evaluation is not good enough, you should consider improving your computer's 3D performance
before e.g. reducing the line of sight in the game options.


Operating Systems

Windows 2000
Though GOTHIC does run under Windows 2000, we recommend playing GOTHIC under Windows 98, since the drivers for  video cards to be used under Windows 2000 are usually of a higher standard and are updated more often.

However, once the Service Pack 2 has been introduced, the current AGP problems ought to be fixed under Windows 2000 as well, so that the performance should become about the same as when running the game under Windows 98.

Windows 95
We recommend updating Windows 95 to Windows 98, since Windows 95 causes problems with AGP support.

Windows NT
GOTHIC requires DirectX 7or higher, therefore it will not run under Windows NT.



Video Cards

When trying to get the best possible performance results, make sure that you have installed the correct AGP drivers for your motherboard. There is an excellent driver support site available for motherboards using the VIA chipset.
Check the VIA Homepage www.viatech.com.
