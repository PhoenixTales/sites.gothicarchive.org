Program Fixes:


- Improved loading times for saved games.

- Improved saving times for saved games.

- Voodoo3 display errors have been eliminated. This was a problem not only experienced with Voodoo3, but with all video cards whose maximum texture dimensions were less than or equal to 256 when displaying texts and menus.

- Deactivating the sky effects in the settings now also deactivates visualization of the Magic Barrier.

- Characters sometimes moved across the screen while the game was paused (e.g. in menus, in the journal or the status screen). This will now no longer occur.

- The focus names for objects were sometimes displayed in the wrong place. This has been fixed.

- The rendering performance has been improved slightly.

- The performance in forests has been improved slightly.

- Possible crash errors have been fixed.

- Fight control (e.g. parry) has been improved.

- Map screen can now be quitted by pressing the "M" key.

- Loss of talents has been fixed.

- Loading time for saved games in the same level has been improved through visual caching.

- A crash error when quitting the game has been fixed.

- Using the winch in front of the Free Mine sometimes had no effect. This has been fixed.

- NPC perception has been optimized.

- NPC recognition of ladders has been improved (particularly in the Old Mine).

- Animations played twice at the start of the dialogue have been fixed.

- Quickly quitting dialogues now also exits dialogue animations.

- The smith sometimes didn't have a hammer in his hand ...

- Torches no longer stay lying around forever.

- The menu can now be activated at most times, though the "Save" option may be missing (e.g. during transformation).

- The startup speed (when clicking on "New Game") has been improved (by about 8 seconds).

- Chapter transitions are introduced by a special image.

- The "Model Details" slider has been corrected. (The maximum is now on the right and the minimum on the left, instead of the other way around.)

- TestMode can no longer be activated in gothic.ini.

- The game clock is now displayed in the Save Game menu.

- Camera mode "Default Mob Interact" improved.

- In the dialogue view, the usual first person view sometimes caused the player character to be invisible. This has been fixed.

- Dialogue camera preset: In some cases, only the speaker's ear was in view. This has been fixed.

- Boot error under Win95: User32.DLL: BlockInput() not found. This has been fixed.

- Two spell crashes have been fixed.

- A slight reverb effect has been added for indoor locations.

- Sound performance slightly improved for high polyphonie.

Hints and Troubleshooting:


- Old saved games can still be loaded.

- Gothic does not support the Windows 95 operating system.

- For video cards from NVidia (particularly TNT, TNT2, GeForce) the "Fog Table Emulation" must be activated in the driver settings. To do this, click on the following options in the Windows start menu: "Control Panel - Display - GeForce - Advanced ... - Direct3D". Tick the box next to the entry "Enable Fog Table Emulation".

- Using certain drivers in combination with some video cards from ATI (particularly Radeon) can lead to problems with the fog. This will lead to the whole screen being filled with just one colour (e.g. brown, pink or black) whenever the game is launched. This is a driver problem that ATI should solve for you. As a short-term solution, the fog effect used in Gothic can be deactivated by editing the .ini file. Use a text editor to open the file "Gothic\System\Gothic.ini" and reset the entry "zFogDisabled=0" to read "zFogDisabled=1".

- If Gothic causes problems with any hardware configuration, the problem may be narrowed down by deactivating the sound effects and/or the background music. To do this, open the file "Gothic\System\Gothic.ini" with a text editor and edit the entries "soundEnabled=1" and "musicEnabled=1" (replace the 1 with a 0), which means that the sound/music will be deactivated right from the very beginning of the game. If this solves the problem, it could mean that you are running the game with an outdated or defect sound card or a sound card with faulty drivers.

- Addenda to the GoldMaster version manual/readme: In the Trade dialogue, items can be moved from one category to another by using the key combinations TURN_LEFT+CTRL and TURN_RIGHT+CTRL. Additionally, ten items can be moved at once by using the key combination STRAFE_LEFT+CTRL and STRAFE_RIGHT+CTRL. This speads up the transfer of multiple units.
